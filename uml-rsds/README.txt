The CRA algorithms are implemented by GUI.class, execute as 
java GUI

Option loadModel loads a model file from in.txt (eg., inA.txt represents the test case A, etc). 
Option createClasses or createClasses1 defines an initial assignment of features to classes,
refactor and cleanup can be used for manual improvement of this assignment. 
 
initialise creates a genetic algorithm representation, and evolve followed by nextgeneration
executes one iteration of the GA. postprocess copies the best individual to a class model.
